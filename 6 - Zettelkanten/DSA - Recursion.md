# DSA - Recursion

*Recursion is a programming technique where a function calls itself to solve smaller instances of the same problem.* 

It is a powerful tool in computer science, particularly in the context of data structures and algorithms. Recursion is based on the concept of **divide and conquer**, breaking complex problems into simpler subproblems until reaching a base case that can be solved directly.
