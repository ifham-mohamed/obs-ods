# ITPM - Assignment

To evaluate the performance of the four companies (A, B, C, D) in manufacturing chairs, we can define several key performance parameters:

### Performance Parameters
1. **Production Rate**: Number of chairs produced per week.
2. **Cumulative Production**: Total number of chairs produced up to that week.
3. **Cost Efficiency**: Cost incurred per chair produced (Total cost / Total chairs produced).
4. **Projected Completion**: Estimated number of chairs to be produced by the end of the project based on current trends.
5. **Variance from Target**: Difference between the target production and actual production.
